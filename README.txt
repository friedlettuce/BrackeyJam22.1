Notes
- The population is less likely to purchase VR if they're generally happy

At the beginning of each day:
- Inventory purchased, servers do not need to be rebought
- Each server is maintained with a fee

During the day:
- NPCs decide to buy or not
- New users take server capacity and 1 hardware away

At the end of every day before inventory stocking:
- Users quit based on number of software available
- Each active user pays a fee

Game Over:
- High score is determined by user base and money
